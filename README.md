# SONGO - Jeu de semailles traditionnel

## Description
Songo est un jeu de semailles pratiqué originellement par les Ekang du Cameroun, du Gabon et de Guinée équatoriale. Chaque joueur possede 7 cases contenant 5 graines au depart.

## Fonctionnalites
- Mode 2 joueurs local
- Mode contre IA (3 niveaux : Facile, Moyen, Difficile)
- Interface moderne avec animations 3D
- Pause / Reprise
- Abandon avec enregistrement victoire
- Historique des captures
- Statistiques par adversaire
- Sauvegarde des scores (localStorage)
- Export JSON de l'historique
- Deux themes (Bois / Moderne)
- Chronometre par partie

## Regles du jeu
1. Chaque joueur possede 7 cases avec 5 graines
2. Ramasser toutes les graines d'une case de son territoire
3. Les deposer une a une sans sauter de case
4. Distribution : droite vers gauche (son camp) puis gauche vers droite (camp adverse)
5. Capture : si derniere graine tombe dans case adverse contenant 1 a 3 graines
6. Solidarite : si camp adverse vide, donner au moins 7 graines
7. Interdit : semer 1 ou 2 graines depuis sa case 7
8. Interdit : vider completement le camp adverse
9. Victoire : 40 graines ou moins de 10 graines restantes
